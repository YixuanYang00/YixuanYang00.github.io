# Yixuan Yang's Academic Homepage

This repository contains the source code for my personal academic homepage.

## Contents

- `index.html`: main homepage
- `css/`: page styles
- `imgs/`: profile photo and image assets
- `files/`: CV and other downloadable files

## GitHub Pages

This site is hosted with GitHub Pages. Because the repository is named:

```text
YixuanYang00.github.io
```

the homepage is available at:

```text
https://yixuanyang00.github.io/
```

## Local Preview

Open `index.html` directly in a browser, or run a local server from the repository root:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
